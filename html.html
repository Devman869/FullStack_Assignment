HTML Overview

HTML (HyperText Markup Language) is the standard markup language used to create the structure of webpages. It defines the elements and their content, giving meaning to the structure of a page.

Basic HTML Structure

Here is the basic structure of an HTML document:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Page Title</title>
  </head>
  <body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

Common HTML Tags

HTML consists of various tags that define different types of content. Here are some commonly used tags:

  • <h1>, <h2>, <h3>...<h6> - Header tags, used to define headings.
  • <p> - Paragraph tag, used for text content.
  • <a> - Anchor tag, used for links.
  • <ul> <ol> - Unordered and ordered lists.
  • <img> - Image tag, used to display images.
  • <div> - A container element used for grouping content.
  • <span> - Inline container used to style a portion of text.

HTML Forms

Forms are used to collect user input. Here is an example of a basic form:

<form action="submit.html" method="POST">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name"><br>
  <label for="email">Email:</label>
  <input type="email" id="email" name="email"><br>
  <input type="submit" value="Submit">
</form>

Semantic HTML

Semantic HTML refers to using HTML tags that convey meaning about the content they contain. Examples of semantic HTML tags include:

  • <header> - Represents the header of a page or section.
  • <footer> - Represents the footer of a page or section.
  • <article> - Represents an independent section of content.
  • <section> - Represents a section of content within the page.
  • <nav> - Represents navigation links.

Embedding Media in HTML

HTML allows you to embed images, videos, and audio in webpages using the following tags:

  • <img> - Embeds images.
  • <video> - Embeds video files.
  • <audio> - Embeds audio files.

Here’s an example of embedding a video:

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
  Your browser does not support the video tag.
</video>

HTML Tables

Tables are used to display data in a structured format. Here’s an example:

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Jane</td>
    <td>30</td>
  </tr>
</table>

HTML5 New Features

HTML5 introduced new tags and features such as:

  • <article> - For independent content pieces.
  • <aside> - For content tangentially related to the main content.
  • <section> - For grouping content into sections.
  • <video> - For embedding video content.
  • <audio> - For embedding audio content.
  • <canvas> - For rendering 2D graphics.
