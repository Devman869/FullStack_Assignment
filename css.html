CSS Overview

CSS (Cascading Style Sheets) is used to style and layout web pages, such as changing fonts, colors, spacing, and positioning elements.

CSS Selectors

CSS selectors target HTML elements to apply styles. Here are some basic selectors:

  • .class-name targets elements with a specific class.
  • #id-name targets an element with a specific ID.
  • element targets all elements of a specific type.

Example:

.class-name {
  background-color: yellow;
}

Above, the selector .class-name applies a yellow background to any element with the class class-name.

The CSS Box Model

The CSS box model is a fundamental concept in web design. It describes the rectangular boxes generated for elements, which include padding, border, and margin.

This is an element with a border, padding, and defined width/height.

The box model consists of:

  • Content: The actual content, like text or images.
  • Padding: Space between content and border.
  • Border: A line around the padding (optional).
  • Margin: Space between the border and surrounding elements.

Flexbox

Flexbox is a layout model that allows for easy alignment and distribution of elements within a container. It is used to create flexible and responsive layouts.

Flexbox Properties:

  • justify-content aligns items horizontally (left, center, right, space-between, etc.).
  • align-items aligns items vertically (top, middle, bottom, etc.).
  • flex-direction sets the direction of flex items (row, column).

Responsive Design

Responsive design ensures that web pages look good on all devices, from mobile phones to desktops. CSS media queries allow you to apply styles based on the device's screen size.

@media (max-width: 768px) {
  body {
    background-color: lightgray;
  }
}

The example above changes the background color to light gray when the viewport width is 768px or less (usually for tablets and smaller devices).

Frequently Asked Questions (FAQs)

Flexbox is a one-dimensional layout model (works in rows or columns), whereas CSS Grid is two-dimensional (works with both rows and columns). Flexbox is great for simpler layouts, while CSS Grid is ideal for more complex designs.

You can center elements using Flexbox by setting justify-content: center; and align-items: center;. Alternatively, for block elements, you can use margin: 0 auto; to horizontally center them.
